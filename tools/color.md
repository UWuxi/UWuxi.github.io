# 🎨 随机颜色

生成随机颜色，一键复制 CSS 代码。

<ColorGenerator />

<script setup>
import ColorGenerator from './components/ColorGenerator.vue'
</script>

## 使用说明

- 点击"生成颜色"按钮随机生成
- 点击色块复制 CSS 代码
- 支持 HEX、RGB、HSL 格式

## CSS 颜色格式

| 格式 | 示例 | 说明 |
|------|------|------|
| HEX | `#5f67ee` | 最常用，6位16进制 |
| RGB | `rgb(95, 103, 238)` | 红绿蓝三通道 |
| HSL | `hsl(238, 80%, 65%)` | 色相、饱和度、亮度 |
